Мы создаем инфраструктуру по шаблонам. Есть 2 основных понятия — «стек» и «шаблон». Стеком (англ. stack) называется набор облачных ресурсов (машин, логических томов, сетей и т.д.), объединённых в цельную структуру. Шаблон — это описание стека. У нас есть гит репозиторий, где мы храним все шаблоны. Там мы описываем ресурсы, и их связи и зависимости. Они доступны для любого человека в компании с возможностью pull request --- layout: false ## Инфраструктура по требованию Создание стеков из шаблонов ```bash $ heat stack-create TEST -f test.yml \ -P key_name="test" \ -P server_name=Centos7 \ -P server_flavor=medium \ -P server_image="CentOS 7 64-bit" ``` ???
